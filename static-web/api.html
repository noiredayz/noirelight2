PogChamp

Usage:

  • GET /bot/api/pogchamp
  • GET /bot/api/pogchamp?date=MMMM-DD-YY

    • Return values:

    API always returns application/json
    If not date is specified it returns the last record.
    Twitch usually adds new PogChamp around 18:00 CET/CEST (12:00 pm EST, 9:00 am PST, 4:00am AEDT),
    then tweets about it usually within a half hour. Records are added after I can confirm
    the identity of the person on the new emote and the person or Twitch tweeted about it.
    On errors and invalid applications you get:
    {error : "error description}
    On success you get:
    {
    "emote":"fully twitch URL to the 3x size of the mote",
    "date":"YYYY-DD-MM",
    "comment":"name, twitch name, twitter name and a few words about the streamer",
    "sourceLinks":"twitter link(s), separate by space if more"
    }
